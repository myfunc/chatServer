# chatServer
Installation:
1) Install node.js server
2) Windows:
    Run "run.bat" 
  or
    cmd:
      node server.js
3) Server is running on port 80
4) http://localhost/
  and 
    http://localhost/jelly 
    is a canvas bubble test. Check it!

Require modules on Node.js

1) NodeStatic module
install:
  npm install node-static -g
